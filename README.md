**Quantizer** is a React-based web interface designed to act as a sleek, modern GUI wrapper for FFmpeg. It focuses on converting native macOS formats (like `.mov`) to universal formats (like `.mp4`) while providing a stunning visual experience.

### Features
*   **3D Landing Page**: An interactive 3D particle visualization (Torus Knot or Sphere).
*   **Drag & Drop**: Intuitive file selection.
*   **File Analysis**: Displays metadata such as size, type, and extension.
*   **Conversion Matrix**: Full control over containers, codecs (H.264, H.265, ProRes), CRF, and presets.
*   **Command Generation**: Generates the optimized FFmpeg command line for you to execute in your terminal.
*   **Filename Sanitization**: Automatically replaces spaces and special characters in the output filename with underscores.

### Usage
1.  Click "Quantify File" on the home screen.
2.  Drop a video or image file into the drop zone.
3.  Adjust settings in the "Conversion Matrix".
4.  Click "Generate Command" and copy the script to your clipboard.
5.  Paste into your macOS Terminal to execute the conversion.
